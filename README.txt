# Hand Gesture Recognition Project

This repository contains Python scripts for hand gesture recognition using computer vision and webcam input.

## Files

- `01_webcam_test.py`: A simple webcam test script.
- `02_hand_landmarks.py`: Detects hand landmarks using MediaPipe.
- `03_hand_gesture_recognition.py`: Recognizes hand gestures based on landmarks.
- `04_gesture_game_control.py`: Uses hand gestures to control a game.

## Requirements

- Python 3.x
- OpenCV
- Mediapipe
- Other dependencies as needed
